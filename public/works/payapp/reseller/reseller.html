영업방법

  • 판매점 가입링크를 활용한 온라인 영업
    리셀러 가입 시 자동으로 생성되는 리셀러별 판매점 가입 전용링크를 각종 사이트, SNS 등에 배너, URL 형태로 게시해 판매점 모집 가능
  • 웹 관리자에서 판매점 직접 등록
    리셀러 전산에서 판매점 정보를 입력해 리셀러가 판매점 계정을 직접 생성 가능
  • 회원가입 API 연동
    리셀러의 판매자 등록에 관한 페이앱 API 연동을 통해 플랫폼사/서비스가 원하는 방식으로 자체 시스템에 페이앱 판매점 등록 기능 접목 가능
